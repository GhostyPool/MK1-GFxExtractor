# MK1-GFxExtractor
A Python script for extracting Scaleform GFx data from Mortal Kombat 1 .uasset files. Extract .uasset files from the game using [FModel](https://github.com/4sval/FModel) or [UModel](https://www.gildor.org/en/projects/umodel).

May or may not work with other Unreal Engine 4/5 games using Scaleform GFx.

## Usage
```bash
python MK1-GFxExtractor.py <input> <output>
